# SEO
SEO (Scorpion Email Osint) es una herramienta de osint para un correo electronico.

# Uso e Instalacion:
**Kali Linux / Parrot OS / Termux**
```bash
git clone https://github.com/Euronymou5/SEO
```
```bash
cd SEO
```
```bash
pip3 install -r requirements.txt
```
```bash
python3 seo.py
```
