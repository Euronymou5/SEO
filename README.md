# Uso e Instalacion:
**Kali Linux / Parrot OS / Termux**

**Compatibilidad con Black Arch Linux!**
```bash
git clone https://github.com/Euronymou5/SEO
```
```bash
cd SEO
```
```bash
pip3 install -r requirements.txt
```
```bash
python3 seo.py
```
